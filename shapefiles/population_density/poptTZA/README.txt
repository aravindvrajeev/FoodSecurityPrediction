AfriPop (www.afripop.org) dataset details
_______________________

DATASET: Alpha version 2010 and 2015 estimates of numbers of people per grid square, with national totals adjusted to match UN population division estimates (http://esa.un.org/wpp/) and remaining unadjusted.
REGION: Africa
SPATIAL RESOLUTION: 0.000833333 decimal degrees (approx 100m at the equator)
PROJECTION: Geographic, WGS84
UNITS: Estimated persons per grid square
MAPPING APPROACH: Random Forest
FORMAT: Geotiff (zipped using 7-zip (open access tool): www.7-zip.org)
FILENAMES: Example - AGO_popmap10adj_v2b.tif = Angola (AGO) population count map for 2010 (10) adjusted to match UN national estimates (adj), version 2b (v2b). Population maps are updated to new versions when improved census or other input data become available.
DATE OF PRODUCTION: July 2013

Also included: (i) Google Earth file
